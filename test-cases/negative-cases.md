# Негативные тест-кейсы

## TC-N01: Ввод букв в поле суммы

**ID:** TC-N01  
**Название:** Проверка обработки нечислового ввода.

**Предусловия:**  
- Открыта страница конвертера.  
- Списки валют загружены.

**Шаги:**  
1. В поле «Сумма» ввести `abc`.  
2. В выпадающем списке «Из валюты» выбрать USD (если не выбрано по умолчанию).  
3. В выпадающем списке «В валюту» выбрать EUR.  
4. Нажать кнопку «Конвертировать».

**Ожидаемый результат:**  
Под полем результата появляется сообщение «Введите корректную сумму».

---

## TC-N02: Ввод отрицательного числа

**ID:** TC-N02  
**Название:** Проверка обработки отрицательного значения.

**Предусловия:**  
- Открыта страница конвертера.  
- Списки валют загружены.

**Шаги:**  
1. В поле «Сумма» ввести `-50`.  
2. В выпадающем списке «Из валюты» выбрать USD.
3. В выпадающем списке «В валюту» выбрать EUR.
4. Нажать кнопку «Конвертировать».

**Ожидаемый результат:**  
Конвертация выполняется, отображается результат с отрицательным знаком.  
(Это поведение допустимо для математической функции, но с точки зрения бизнес-логики может считаться багом – при желании оформить баг-репорт.)

## TC-N03: Отсутствие интернета

**ID:** TC-N03  
**Название:** Проверка поведения при отсутствии соединения.

**Предусловия:**  
- Открыта страница конвертера (загружена до отключения интернета).

**Шаги:**  
1. В DevTools перейти в режим Offline.  
2. Перезагрузить страницу.

**Ожидаемый результат:**  
Появляется сообщение «Ошибка загрузки курсов валют. Проверьте подключение к интернету.».
