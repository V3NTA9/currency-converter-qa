# Позитивные тест-кейсы

## TC-01: Конвертация USD в EUR

**ID:** ТC-01
**Название:** Проверка конвертации из доллара США в евро.

**Предусловия:**
- Открыта страница конвертера.
- Списки валют загружены.

**Шаги:**
1. В поле «Сумма» ввести 100.
2. В выпадающем списке «Из валюты» выбрать USD.
3. В выпадающем списке «В валюту» выбрать EUR.
4. Нажать кнопку «Конвертировать».

**Ожидаемый результат:**
Отображается результат вида `100 USD = XX.XX EUR` (где XX.XX - рассчитанное значение).

## TC-03: Конвертация после изменения валюты в списке «Из валюты».

**ID:** ТС-03
**Название:** Проверка конвертации после изменения валюты в списке «Из валюты».

**Предусловия:**
- Открыта страница конвертера.
- Списки валют загружены.
- В поле «Сумма» введено 100.
- В выпадающем списке «Из валюты» выбрана валюта USD.
- В выпадающем списке «В валюту» выбрана валюта EUR.
- Нажата кнопка «Конвертировать», получен результат (например, `100 USD = 86.00 EUR`).

**Шаги:**
1. В выпадающем списке «Из валюты» выбрать другую валюту, например, AUD.
2. Нажать кнопку «Конвертировать».

**Ожидаемый результат:**  
Результат обновляется и отличается от предыдущего: отображается `100 AUD = XX.XX EUR`. Сумма в EUR пересчитана согласно курсу AUD/EUR.
