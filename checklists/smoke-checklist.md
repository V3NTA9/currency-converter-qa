# Smoke-чеклист для конвертера валют

1. Открытие страницы — страница загружается без ошибок.
2. Отображение выпадающих списков — в списках «Из валюты» и «В валюту» присутствуют валюты.
3. Выбор валют — можно выбрать любую валюту из списка.
4. Ввод суммы — поле ввода принимает числа.
5. Конвертация — при нажатии на кнопку результат отображается корректно.
6. Обработка ошибок ввода — при вводе букв появляется сообщение «Введите корректную сумму».
7. Смена валют — после смены валют результат пересчитывается.
